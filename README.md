# 4-bit ALU using Verilog

This project implements a simple 4-bit Arithmetic Logic Unit (ALU).

## Operations
- Addition
- Subtraction
- AND
- OR
- XOR

## Files
alu.v → ALU design  
alu_tb.v → Testbench for simulation

## Tools Used
Verilog HDL  
ModelSim / Cadence simulation
